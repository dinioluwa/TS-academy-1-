# TS-academy-1-
 Titanic Dataset EDA  Exploratory Data Analysis of the Titanic dataset. Investigates survival patterns across passenger class, age, gender, and fare using Python libraries such as Pandas, Matplotlib, and Seaborn to analyze and visualize patterns within the dataset.

This notebook presents a comprehensive Exploratory Data Analysis (EDA) of the famous Titanic dataset. The analysis dives into the factors that influenced passenger survival during the tragic sinking of the RMS Titanic in 1912.
 Objectives

Explore the demographic and socioeconomic profile of passengers
Identify key factors that influenced survival rates
Visualize relationships between features such as age, gender, class, and fare
Handle missing data and prepare features for potential modeling

 Tools & Libraries

Python
Pandas – data manipulation
Matplotlib & Seaborn – data visualization
NumPy – numerical operations
Jupyter Notebook – interactive analysis

Key Insights

Survival rate breakdown by passenger class, gender, and age group
Fare and embarkation point analysis
Family size and its effect on survival probability
